       - Extract value with `val['#text'] ?? val` pattern for elements with attributes

       - Sum budgets by type="1", transactions by type codes "2" and "3"
       - Include negative transaction values in sums (they represent valid reversals)
       - `normalizedImplementerName`: lowercase + trim of implementerName
       - `rawData` JSONB: store only structured key fields (title, implementer, funder, country, status), NOT full XML
