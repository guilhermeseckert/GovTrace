```typescript
import type { SankeyNode as D3SankeyNode, SankeyLink as D3SankeyLink } from 'd3-sankey'

type SankeyNodeData = { id: string; name: string; type: 'entity' | 'party' | 'department' }
type SankeyLinkData = { source: string; target: string; value: number }
type ComputedNode = D3SankeyNode<SankeyNodeData, SankeyLinkData>
type ComputedLink = D3SankeyLink<SankeyNodeData, SankeyLinkData>
```

**Layout constants:**
```typescript
const MARGIN = { top: 20, right: 20, bottom: 20, left: 20 }
const HEIGHT = 400
```

**Data fetch:** `useEffect` on `entityId` → call `getMoneyFlow({ data: { id: entityId } })` → store raw `nodes` and `links` in state. Handle loading and error states.

**Layout computation (inside useMemo, depends on `[rawNodes, rawLinks, width]`):**
- Deep-copy before passing to sankey (Pitfall 4): `nodes.map(n => ({ ...n }))`, `links.map(l => ({ ...l }))`
- Build layout:
```typescript
const sankeyLayout = sankey<SankeyNodeData, SankeyLinkData>()
  .nodeWidth(20)
  .nodePadding(12)
  .nodeId(d => d.id)
  .nodeAlign(sankeyLeft)
  .extent([[MARGIN.left, MARGIN.top], [width - MARGIN.right, HEIGHT - MARGIN.bottom]])

const { nodes: computedNodes, links: computedLinks } = sankeyLayout({
  nodes: rawNodes.map(n => ({ ...n })),
  links: rawLinks.map(l => ({ ...l })),
})
```

**Empty state:** If `rawNodes.length < 3`, render `<p className="...">{en.viz.sankey.emptyLabel}</p>` — do not run sankeyLayout.

**Color scheme by node type:**
- `entity` → `getColor('--primary')`
- `party` → `getColor('--accent-foreground')`
- `department` → `getColor('--ring')`

**SVG rendering (D3 for math, React for rendering — no d3.select DOM mutation):**
- Outer `<div ref={containerRef} className="relative w-full">` with height={HEIGHT}
- SVG `width={width} height={HEIGHT}` with `role="img"` and `aria-label={en.viz.sankey.ariaLabel}`
- For each computed node: `<rect x={node.x0} y={node.y0} width={node.x1 - node.x0} height={node.y1 - node.y0} fill={nodeColor} rx={2} />`
- Below each node rect: `<text x={node.x1! + 4} y={(node.y0! + node.y1!) / 2} dominantBaseline="middle" fontSize={11} fill="currentColor">` with node.name truncated to 20 chars
- For each computed link: `<path d={sankeyLinkHorizontal()(link) ?? ''} fill="none" stroke={sourceNodeColor} strokeWidth={Math.max(1, link.width ?? 0)} strokeOpacity={0.4} />`
- Link color = source node color (research spec: "source node color at 50% opacity" — use strokeOpacity 0.5 on the path)

**Loading state:** `<div className="flex h-[400px] items-center justify-center text-muted-foreground">{en.viz.sankey.loadingLabel}</div>`

**Ultracite compliance:** No `any`, no `console.log`, arrow functions, `const`, no barrel imports.

**Layout constants:**
```typescript
const MARGIN = { top: 30, right: 20, bottom: 50, left: 50 }
const HEIGHT = 320
const AXIS_Y = 160  // center Y for the timeline axis line
const MARKER_RADIUS = 6
```

**Data fetch:** `useEffect` on `entityId` → `getTimeline({ data: { id: entityId } })` → store events in state.

**Scale computation (inside useMemo):**
- Parse event dates: `new Date(event.date)` — skip invalid dates
- `d3.scaleTime().domain([minDate, maxDate]).range([MARGIN.left, scrollWidth - MARGIN.right])`
- `scrollWidth = Math.max(width, events.length * 12 + MARGIN.left + MARGIN.right)` — ensures scrollable space

**Zoom for horizontal scroll (attach via `useEffect([], [])` — Pitfall 2 applies here too):**
- Attach `d3.zoom().scaleExtent([1, 1]).translateExtent([[0, 0], [scrollWidth, HEIGHT]])` with `translateExtent` constraining to X-only scroll
- On zoom event: update `translateX` state (only X component of `event.transform.x`)
- Attach to outer `<div>` using `d3.select(containerRef.current)`

**SVG rendering:**
- Container: `<div ref={containerRef} className="overflow-hidden" style={{ height: HEIGHT }}>` — let d3.zoom handle scroll via transform
- SVG: `width={scrollWidth} height={HEIGHT}` with `role="img"` `aria-label={en.viz.timeline.ariaLabel}`
- Apply X-translate: `<g transform={`translate(${translateX}, 0)`}>`
- **Axis line:** horizontal `<line x1={MARGIN.left} x2={scrollWidth - MARGIN.right} y1={AXIS_Y} y2={AXIS_Y} stroke="currentColor" strokeOpacity={0.2} />`
- **Election year lines:** For each year in ELECTION_YEARS where date falls within domain: `<line x1={xScale(new Date(year, 0, 1))} x2={...} y1={MARGIN.top} y2={HEIGHT - MARGIN.bottom} stroke="currentColor" strokeOpacity={0.15} strokeDasharray="4,4" />` with `<text>` label at top showing the year
- **Event markers by shape:**
  - `circle`: `<circle cx={x} cy={AXIS_Y} r={MARKER_RADIUS} fill={color} />`
  - `square`: `<rect x={x - MARKER_RADIUS} y={AXIS_Y - MARKER_RADIUS} width={MARKER_RADIUS * 2} height={MARKER_RADIUS * 2} fill={color} />`
  - `diamond`: `<polygon points={`${x},${AXIS_Y - MARKER_RADIUS} ${x + MARKER_RADIUS},${AXIS_Y} ${x},${AXIS_Y + MARKER_RADIUS} ${x - MARKER_RADIUS},${AXIS_Y}`} fill={color} />`
  - `triangle`: `<polygon points={`${x},${AXIS_Y - MARKER_RADIUS} ${x + MARKER_RADIUS},${AXIS_Y + MARKER_RADIUS / 2} ${x - MARKER_RADIUS},${AXIS_Y + MARKER_RADIUS / 2}`} fill={color} />`
- **Hover tooltip on marker:** `onMouseEnter` → set `hoveredEvent` state → render absolute positioned div with date, description, amount (if present), eventType label
- **X-axis ticks:** Use `xScale.ticks(d3.timeYear.every(2))` to get year tick positions, render `<text>` labels below axis line
- **Legend:** Render below SVG a flex row of colored squares + type labels from EVENT_CONFIG

**Empty state:** If `!loading && events.length === 0`, render `en.viz.timeline.emptyLabel`.
