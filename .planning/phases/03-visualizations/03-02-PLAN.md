**Component structure — useNetworkGraph custom hook pattern to isolate simulation logic:**

```typescript
function useNetworkGraph(entityId: string, activeTypes: string[]) {
  const [nodes, setNodes] = useState<GraphNode[]>([])
  const [links, setLinks] = useState<GraphLink[]>([])
  const [loading, setLoading] = useState(true)
  const [truncated, setTruncated] = useState(false)
  const simulationRef = useRef<d3.Simulation<GraphNode, GraphLink> | null>(null)

  const loadData = useCallback(async (expandId: string, existingNodes: GraphNode[]) => {
    setLoading(true)
    const data = await getGraphData({ data: {
      id: expandId,
      depth: 1,
      connectionTypes: activeTypes.length > 0 ? activeTypes : undefined,
    }})

    // Merge: preserve x/y/vx/vy for already-rendered nodes (Pitfall 6)
    const positionMap = new Map(existingNodes.map(n => [n.id, { x: n.x, y: n.y, vx: n.vx, vy: n.vy }]))
    const merged = data.nodes.map(n => ({
      ...n,
      ...(positionMap.get(n.id) ?? {}),
    }))

    setNodes(merged)
    setLinks(data.edges.map(e => ({ ...e, source: e.sourceId, target: e.targetId })))
    setTruncated(data.truncated)
    setLoading(false)
  }, [activeTypes])

  // Initial load
  useEffect(() => { loadData(entityId, []) }, [entityId, loadData])

  // Simulation — restart whenever nodes/links change
  useEffect(() => {
    if (nodes.length === 0) return
    simulationRef.current?.stop()

    const sim = d3.forceSimulation<GraphNode>(nodes)
      .force('link', d3.forceLink<GraphNode, GraphLink>(links).id(d => d.id).distance(80))
      .force('charge', d3.forceManyBody<GraphNode>().strength(-200))
      .force('center', d3.forceCenter(400, 300))
      .force('collision', d3.forceCollide<GraphNode>(20))

    // Pitfall 1: use function form to avoid stale closure
    sim.on('tick', () => {
      setNodes(prev => [...sim.nodes()])
      setLinks(prev => [...(links as GraphLink[])])
    })

    simulationRef.current = sim
    return () => { sim.stop() }
  }, [nodes.length, links.length]) // deliberately use .length to avoid infinite loops

  const expandNode = useCallback((nodeId: string) => {
    loadData(nodeId, nodes)
  }, [loadData, nodes])

  return { nodes, links, loading, truncated, expandNode }
}
```

**Main component — zoom via useEffect with [] (Pitfall 2: stable SVG, never conditionally render):**
- Container: `<div ref={containerRef} className="relative w-full" style={{ height: '500px' }}>`
- Outer `<svg>` ref gets d3.zoom attached in `useEffect([], [])` (mount-only). Zoom range: `[0.1, 8]`.
- Transform stored in `useState<{ x: number; y: number; k: number }>({ x: 0, y: 0, k: 1 })`.
- Inner `<g transform={`translate(${t.x},${t.y}) scale(${t.k})`}>` renders all edges then all nodes.
- `role="img"` on SVG, `aria-label={en.viz.graph.ariaLabel}`.

**Rendering edges:**
- `stroke-dasharray`: `donor_to_party` = none (solid), `vendor_to_department` = "6,3" (dashed), `lobbyist_to_official` = "2,4" (dotted), all others = none
- `stroke-width`: `Math.log1p(link.transactionCount) * 1.5` capped at 8px
- `stroke-opacity`: 0.5
- Edge `source` and `target` are resolved GraphNode objects after simulation runs — access `(link.source as GraphNode).x` with type assertion

**Rendering nodes:**
- Circle `r={depth === 0 ? 16 : 10}` (root node larger)
- Fill from `nodeColors[node.entityType]()` using useChartColors
- `onClick`: call `expandNode(node.id)`
- `onMouseEnter`: set tooltipState `{ node, x: event.clientX, y: event.clientY }`
- `onMouseLeave`: set tooltipState null
- `aria-label={en.viz.graph.expandNode + ': ' + node.name}`
- Node label: `<text>` below circle with `dy="1.4em"`, `textAnchor="middle"`, `fontSize={depth === 0 ? 12 : 10}`, `fill="currentColor"`, truncated to 15 chars with ellipsis

**Tooltip — absolute div outside SVG, positioned with `style={{ left: x + 12, top: y - 20 }}`:**
- Show: name, entityType, transactionCount sum for edges connected to this node, totalValue sum

**Filter panel — rendered above SVG:**
- Extract unique `connectionType` values from all links
- Render checkbox `<button>` (toggle) for each type — styled like shadcn Badge toggle
- `activeTypes` state: `useState<string[]>([])` — empty = show all
- On type toggle: update activeTypes, which triggers the `loadData` useEffect

**Loading state:** Show loading spinner overlay (position absolute, centered) OVER the stable SVG (not instead of it — Pitfall 2).

**Truncation warning:** If `truncated`, render amber alert `<div role="alert">` below SVG with `en.viz.graph.truncatedWarning`.

**Empty state:** If `!loading && nodes.length === 0`, render centered empty state inside the SVG container with `en.viz.graph.emptyLabel`.

**Accessibility:** SVG has `role="img"` and `aria-label`. Filter buttons have `aria-pressed`. Loading overlay has `aria-live="polite"`.

**Ultracite compliance:** No `any` types (use GraphNode/GraphLink intersection types). No `console.log`. No barrel imports. Use `for...of` not `.forEach`. Use `const` everywhere. Arrow functions for callbacks.
