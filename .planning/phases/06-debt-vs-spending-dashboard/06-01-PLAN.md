  <action>
    1. **Schema** — Append `fiscalSnapshots` table to `packages/db/src/schema/raw.ts` (DEBT-01, DEBT-04):
       ```typescript
       export const fiscalSnapshots = pgTable('fiscal_snapshots', {
         id: text('id').primaryKey(),               // SHA256(series + ref_date)
         series: text('series').notNull(),          // 'federal_net_debt' | 'accumulated_deficit' | 'gross_liabilities'
         refDate: date('ref_date').notNull(),       // YYYY-MM-01
         valueMillionsCad: numeric('value_millions_cad', { precision: 15, scale: 2 }),
         sourceTable: text('source_table').notNull(),  // '10-10-0002-01'
         sourceUrl: text('source_url').notNull(),
         ingestedAt: timestamp('ingested_at', { withTimezone: true }).notNull().defaultNow(),
       }, (t) => [
         uniqueIndex('fiscal_snapshots_series_date_idx').on(t.series, t.refDate),
         index('fiscal_snapshots_ref_date_idx').on(t.refDate),
       ])
       ```
       Then run `pnpm --filter @govtrace/db drizzle-kit generate` to create the migration, and `pnpm --filter @govtrace/db drizzle-kit push` to apply it.

    2. **Downloader** — Create `apps/ingestion/src/downloaders/fiscal.ts`. Export `downloadFiscalCsv(destDir: string): Promise<{ localPath: string; fileHash: string }>`.
       - Call Stats Canada WDS: `GET https://www150.statcan.gc.ca/t1/wds/rest/getFullTableDownloadCSV/10100002/en`
       - Parse JSON envelope, extract ZIP URL from `object` field
       - Download ZIP, extract data CSV (entry ending `-eng.csv` but NOT containing `_MetaData`)
       - Write CSV to `destDir`, compute SHA-256 hash, return path and hash
       - Use `adm-zip` for extraction (already in catalog, used by international-aid downloader)

    3. **Parser** — Create `apps/ingestion/src/parsers/fiscal.ts`. Export `parseFiscalCsv(csvPath: string, fileHash: string): FiscalSnapshotRow[]`.
       - Use `papaparse` (already in catalog) to parse the CSV
       - **Critical filter**: Only keep rows where `Central government debt` column equals `A. Federal debt (accumulated deficit)` — the table contains 29+ interleaved series (see Research Pitfall 1)
       - Map each row to `{ id: sha256(series + refDate), series: 'accumulated_deficit', refDate, valueMillionsCad: row.VALUE, sourceTable: '10-10-0002-01', sourceUrl: 'https://www150.statcan.gc.ca/t1/tbl1/en/tv.action?pid=1010000201' }`
       - Also extract rows for `B. Net debt` with series `'federal_net_debt'`
       - Handle SCALAR_FACTOR column — Stats Canada values may be in "millions" or "units"; multiply accordingly
       - Return typed array of FiscalSnapshotRow (matching the Drizzle insert shape)

    4. **Upsert** — Create `apps/ingestion/src/upsert/fiscal.ts`. Export `upsertFiscalSnapshots(rows: FiscalSnapshotRow[]): Promise<number>`.
       - Use Drizzle `db.insert(fiscalSnapshots).values(batch).onConflictDoUpdate()` with conflict target on `id`
       - Batch in groups of 500
       - Return total upserted count

    5. **Runner** — Create `apps/ingestion/src/runners/ingest-fiscal.ts`. Export `runFiscalIngestion(): Promise<void>`.
       - Mirror the `runInternationalAidIngestion` pattern exactly: create ingestion_runs record, download, parse, upsert, update run status
       - Source: `'fiscal_snapshots'`, sourceFileUrl: the WDS URL
       - Log progress to console
       - On error: update ingestion_runs status to 'failed' with error message

    2. **Server functions** (DEBT-01, DEBT-02, DEBT-03, DEBT-04) — Create `apps/web/src/server-fns/dashboard.ts` with three server functions following the `createServerFn` pattern from stats.ts:

       **`getDebtTimeline`** — Returns annual debt + aid data points for the chart:
       - Query `fiscal_snapshots` for December values per year (or latest month for current year) for `accumulated_deficit` series, converting from millions to billions
       - Query `international_aid` with `GROUP BY EXTRACT(YEAR FROM start_date)` for annual committed and disbursed totals, converting to billions
       - Join by year, return array of `DebtAidDataPoint` typed objects
       - Each point includes `sourceDebtUrl` and `sourceAidUrl` for DEBT-04 traceability

       **`getDepartmentBreakdown`** — Returns department spending ranked list:
       - Query `international_aid` with `GROUP BY funding_department`, computing total committed, total disbursed, and project count

       - Return typed object with `currentDebtBillions`, `totalAidBillions`, `aidAsPercentOfDebt`, `debtAsOf` (the date string), source URLs for DEBT-04

       Use `sql` template tag from drizzle-orm for the aggregation queries (same pattern as visualizations.ts). Do NOT use `d3.select` or any client-side code in server functions.
