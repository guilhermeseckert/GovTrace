**Plain English connection cards with color-coded badges and SQL-driven "Did you know?" pattern callouts for entity profiles**

## Accomplishments
- ConnectionCards renders entity connections as readable sentences with color-coded left borders and badges from shared CONNECTION_LABELS
- PatternCallouts surfaces notable observations via 4 SQL pattern detectors: multi-department contracts, high-value connections ($500K+), temporal clustering (lobbying within 90 days of contract), and multi-party donations
- Both components handle loading/error/empty states gracefully with skeleton cards and null rendering

## Decisions Made
- BORDER_COLORS kept separate from CONNECTION_LABELS since Tailwind border-l-* classes differ from bg/text badge classes
- HIGH_VALUE_THRESHOLD set to $500K as proxy for top 1% connections
- Pattern priority ordering: monetary patterns first (high-value, multi-dept), then behavioral (temporal, multi-party)
- Used db.execute with raw SQL for temporal clustering self-join — Drizzle ORM lacks support for this query shape
