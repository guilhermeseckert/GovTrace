# Quick 260406-15u: Public Accounts of Canada Ingestion + Dashboard Summary

**One-liner:** Ingested 15,109 Public Accounts "Expenditures by Standard Object" rows across 14 fiscal years into a new `department_expenditures` table, with a dashboard horizontal-bar visualization showing how federal departments allocate budgets across 13 spending categories.

## What Was Built

### Task 1: Schema + Ingestion Pipeline

- **`departmentExpenditures` table** — 8 columns, composite unique index on `(fiscal_year, org_id, standard_object)`, indexes on `fiscal_year` and `org_name`. No JSONB rawData (all 5 CSV fields captured as typed columns).
- **Migration** — `0007_daily_the_hood.sql` generated and pushed to DB.
- **Downloader** — Direct CSV fetch from open.canada.ca (no ZIP). Returns `{ localPath, fileHash }`.
- **Parser** — BOM strip + PapaParse + Zod validation. Negative `expenditures` values preserved (External/Internal revenues are valid accounting entries). SHA256 ID from `fy_ef + ':' + org_id + ':' + sobj_en`.
- **Upsert** — 500-row batches, deduplicated within batch, `onConflictDoUpdate` on PK.
- **Runner** — Creates `ingestion_runs` record, orchestrates download → parse → upsert → status update, catches and records errors.
- **CLI** — `case 'public-accounts'` added to ingest switch; help text updated.

**Ingestion result:** 15,109 rows, idempotent (same count on re-run).

### Task 2: Dashboard Spending Visualization

- **`getSpendingByCategory` server fn** — Groups `department_expenditures` by `standardObject` for the latest fiscal year (via `MAX(fiscal_year)` subquery). Returns `SpendingCategoryRow[]`. Cached under key `spending-by-category`.
- **`SpendingByCategory` component** — Horizontal bar chart for 13 standard object categories. Positive expenditures shown with proportional primary-colored bars + percentage. Negative revenue categories (External/Internal revenues) shown below a separator in a muted style with `100%` bar width at reduced opacity. Includes source link to open.canada.ca dataset.
- **Dashboard route** — New "Federal Spending by Category" section inserted before "Spending by Department". `useQuery` with 1-hour staleTime. Skeleton while loading.
